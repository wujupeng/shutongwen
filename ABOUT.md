# ShuTongWen IME - About

## 关于书同文输入法

书同文输入法是一款专为 Windows 11 设计的现代化中文输入法，致力于提供流畅、智能的输入体验。

> 书同文，车同轨 —— 秦始皇统一度量衡，我们统一中文输入体验。

### 项目愿景

打造一个开源、高性能、可扩展的输入法平台，成为开发者和用户的首选中文输入工具。

### 核心价值

- **用户体验至上** - 低延迟、高准确率的输入体验
- **开放源代码** - 完全开源，欢迎社区贡献
- **跨平台兼容** - 支持 ARM64 和 x64 架构
- **模块化设计** - 易于扩展和定制

### 技术亮点

1. **Adaptive Pipeline** - 智能自适应处理管道，根据系统状态动态调整
2. **Context Analyzer** - 上下文感知输入，智能理解用户意图
3. **Candidate Ranker** - 智能候选排序，学习用户输入习惯
4. **Telemetry System** - 完善的性能监控和追踪系统
