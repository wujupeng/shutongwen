# 书同文输入法 (ShuTongWen IME) - About

## 关于书同文输入法

书同文输入法是一款专为 Windows 11 设计的现代化中文输入法。

> 书同文，车同轨 —— 秦始皇统一度量衡，我们统一中文输入体验。

### 核心宗旨

**高效 + 稳定** —— 这是我们的生命线

作为操作系统常驻基础设施，我们追求：
> "任何时候都像系统原生一样稳定"

### 项目愿景

打造一个**稳定、高效、智能**的输入法运行时系统，让用户忘记输入法的存在。

### 核心价值

| 价值 | 说明 |
|------|------|
| **稳定性优先** | 功能永远不能破坏稳定性 |
| **延迟一致性** | 输入体验平滑跟手，无卡顿 |
| **用户体验至上** | 让用户感觉像系统原生一样自然 |
| **开放源代码** | 完全开源，欢迎社区贡献 |
| **跨平台兼容** | 支持 ARM64 和 x64 架构 |

### 质量指标

| 指标 | 目标 |
|------|------|
| Candidate Flicker Rate | 越低越好 |
| Caret Drift Rate | 接近 0 |
| Composition Abort Rate | 接近 0 |
| Input Freeze Count | 0 |
| PIL (感知输入延迟) | < 8ms |

### 技术亮点

1. **Watchdog Runtime** - 进程监控和崩溃恢复
2. **Input Replay System** - 输入录制、回放和验证
3. **Golden Path Test** - 关键应用输入测试
4. **Latency Optimizer** - 延迟监控和优化
5. **Adaptive Pipeline** - 智能自适应处理管道
